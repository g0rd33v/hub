How it works

Three steps to get started.

It takes about two minutes from zero to a running bot with a dashboard.

  1. Get a token from @BotFather
    Open Telegram, find @BotFather, send /newbot. Give your bot a name and a username ending in bot. BotFather gives you a token — copy it. That's the only time you need BotFather.
  2. Send the token to @LabsHubBot
    Open @LabsHubBot in Telegram and paste the token into the chat. Hub validates it, creates your project, and links the bot — in seconds. Hub replies with your dashboard link.
  3. Manage everything from the dashboard
    Bookmark your dashboard link. That's your control panel. Broadcast, edit commands, update your bot's profile, view analytics — all from the web, forever. No code, no servers, no BotFather.
All features

Everything Hub does
with your bot.

A complete list of what you can manage from the dashboard after connecting your token.

📢
Broadcast
 Send one message to all your subscribers at once. Supports HTML formatting — bold, italic, links, code. Shows how many messages were delivered and how many were skipped after sending.
💬
Send to user
 Send a direct message to any specific person by their Chat ID or @username. Useful for support, personal notifications, and testing your bot's responses.
✎
Command editor
 Your bot's command menu loads live from Telegram when you open the dashboard. Edit command names and descriptions inline, add new ones, delete old ones. One click saves everything back to Telegram.
👤
Bot profile
 Edit three things without touching BotFather: display name (what users see in chat), short description (shown in Telegram search), and about text (shown on the bot's profile page). All save in one click.
🔗
Webhook control
 See your webhook's live status — current URL, number of pending updates, last error message. Set a new webhook URL, or delete it and switch back to polling mode. One click, zero restarts.
👥
Subscribers
 When users send /start to your bot, they're added as subscribers. /stop removes them. The live count shows in the dashboard. Broadcast uses this list to reach everyone.
📊
Audience analytics
 Collected automatically, no setup needed. Shows total users, events, DAU (7-day and 30-day), languages and countries with visual bars, a peak hours heatmap, activity by day of week, and a command usage ranking — which commands your users actually call.
📋
Runtime logs
 See recent bot activity in the dashboard. Useful when something isn't working — no SSH, no server access needed. Clear logs in one click.
🔎
Live bot info
 Hit refresh to pull your bot's current data from Telegram — username, display name, status. Confirms the token is still valid and the bot is alive.
👳
Public project page
 Your bot gets a permanent public URL at hub.labs.co/yourbot/. Share it with anyone. The page shows the bot's name, a button to open it in Telegram, and instructions for working with the link in Claude.
